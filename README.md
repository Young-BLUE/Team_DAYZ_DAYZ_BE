## ✨ 소개
</br>

<div align="center">
  DAYZ는 사용자가 여러곳을 통해 원데이 클래스를 찾고 예약해야하는 불편함을 해소하고자</br>
만들어진 원데이 클래스 중계 플랫폼입니다.</br>
DAYZ를 통해 원하는 원데이 클래스를 한눈에 조회하고 예약하세요~!
</div>
